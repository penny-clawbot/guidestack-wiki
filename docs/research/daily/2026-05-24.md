## Top 3 Findings

### 1. AI Mode = New Search Default (Not a Tab)
AI Mode is no longer a separate tab or experiment тАФ it's the primary search interface. Google's entire index is now processed through Gemini 3.5 Flash by default. Queries are "more than doubling every quarter." This is the end of "blue links" as the primary result format for most searches.

### 2. llms.txt тАФ Useful Signal, Not a Standard
The llms.txt spec is gaining community adoption but **has not been adopted by any major AI crawler**. Google explicitly confirmed they don't use it. It remains a navigation hint for AI tools, not a blocking mechanism. Robots.txt remains the only crawler control that matters.

### 3. Structured Data Cleanup тАФ Deprecated Types Still in Templates
Google's phased-out structured data types (course info, estimated salary, learning video, special announcement, vehicle listing) тАФ deprecated September 2025 тАФ are likely still in active templates. Practice Problem fully deprecated January 2026. FAQ schema and HowTo remain valuable for AI Overview visibility.
