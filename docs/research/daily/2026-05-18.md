# GEO/SEO Daily Research — 2026-05-18

## Top 3 Findings

### 1. WebMCP Ships in Chrome 146 Canary (Feb 2026)
Google Chrome shipped WebMCP — a browser API co-authored with Microsoft — that lets websites expose structured, agent-accessible actions via declarative HTML markup or a JavaScript imperative API. Instead of agents parsing the DOM or taking screenshots, they can call structured JSON tools (e.g., `make_reservation`, `check_inventory`) directly. Available now behind a flag in Chrome 146 Canary. This is the most significant agentic web standard to date and changes what "agent-ready" means for any site with bookable actions, e-commerce, or B2B services.

**Source:** [Google Chrome Blog — WebMCP EPP](https://developer.chrome.com/blog/webmcp-epp) | [Postman Guide](https://p0stman.com/guides/agentic-web-webmcp-guide-2026) | Published Feb 18, 2026

---

### 2. llms.txt: AI Search Crawlers Ignore It, IDE Agents Use It
Across 500M+ LLM bot traffic events (90-day window), llms.txt is almost never touched by search/answer crawlers — GPTBot, ClaudeBot, PerplexityBot, OAI-SearchBot, Google-Extended all crawl HTML directly. Google's Gary Illyes confirmed in July 2025 that Google doesn't and won't support llms.txt, comparing it to the discredited keywords meta tag. No major LLM provider (OpenAI, Anthropic, Google, Meta, Mistral) has committed to using it as a production signal.

However, IDE agents (Cursor, Claude Code, GitHub Copilot, Windsurf, Cline, Aider) and MCP servers DO fetch it routinely. The file's real value today is B2A (Business-to-Agent) routing — not AI search citations.

**Adoption:** ~10.13% of domains (SE Ranking, 300K sample).
**Source:** [Limy AI — LLMs.txt in 2026: The Full Guide](https://limy.ai/blog/llms.txt-in-2026-the-full-guide) | May 12, 2026

### 3. Google AI Mode Now Has Side-by-Side Web Browsing (Apr 2026)
Google rolled out a new "side-by-side" mode in AI Mode (April 16, 2026) that lets users explore the web while maintaining an AI Mode conversation. The March 2026 core update also completed its rollout and Google expanded AI Mode's restaurant booking feature to 8 new markets. Zero-click data continues: ~93% of AI-mode queries end without a click (up from earlier years as AI Overviews expand).

**Sources:** [TechCrunch — Google AI Mode web browsing](https://techcrunch.com/2026/04/16/google-now-lets-you-explore-the-web-side-by-side-with-ai-mode/) | Apr 16, 2026 | [Reddit SEO Round-up](https://www.reddit.com/r/seogrowth/comments/1sl74gu/seo_news_march_2026_core_update_rollout_is_now/)
