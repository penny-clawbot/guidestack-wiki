## Top 3 Findings

### 1. Google I/O 2026: Search Agents & AI Mode Go Mainstream
Google unveiled its most significant Search overhaul in 25+ years. Key changes:
- **AI Mode** now has 1B+ monthly users, queries doubling quarterly.
- **New intelligent search box** — expands for complex queries, supports multimodal input (text, images, video, Chrome tabs).
- **Search Agents** launching for AI Pro/Ultra subscribers this summer — persistent background agents that monitor topics and notify users.
- **Generative UI** in Search — AI builds custom widgets/apps on-the-fly (e.g., fitness trackers pulling real-time weather/location data).
- **Agentic booking** expanding to local services; Google can now call businesses on your behalf (pet care, home repair, beauty).

**Implication for PSEO:** Content must feed AI systems that reason across sources in real-time. Structured data + conversational content + persistent freshness signals become critical.

### 2. llms.txt Is Officially Dead as a GEO Tool
Evidence is now conclusive (OtterlyAI 90-day study: 0.1% bot traction). Google officially does not support it. The file was invented for developer docs/Coding assistant context, not AI search visibility. Mintlify adopting it for docs hosts doesn't translate to general web visibility.

**Implication for PSEO:** Remove any llms.txt guidance from templates. Reallocate effort to FAQ structured data, HowTo, and Q&A page patterns that AI Overviews actively cite.

### 3. Google Deprecated 7 Structured Data Types (Jan 2026)
Google began phasing out low-use structured data in June 2025, with full removal of several types in January 2026. Affected: **PracticeProblem**, **Dataset**, and others (full list in Stan Ventures / ViserX coverage). Dataset markup now only serves Dataset Search, not general search.

**Implication for PSEO:** Audit existing templates for deprecated schema types. Schema.org v30.0 (March 2026) added `Credential`, `Error` (with `errorCode`), `ConferenceEvent`, `StatisticalVariable`, and EU Digital Product Passport examples.
