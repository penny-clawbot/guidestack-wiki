# PSEO Daily Research — 2026-06-01

## Top 3 Findings

**1. Google I/O 2026: "Search is AI Search" — AI Mode Now Default (May 19, 2026)**
Google declared the most significant overhaul to its search engine in 25 years. The search box now expands dynamically into AI agents that scan and act on web content. AI Overviews now trigger on ~48% of all queries (up 58% YoY). The distinction between "regular search" and "AI Mode" is effectively gone — most searches produce an AI Overview by default. This collapses the old two-track SEO model.

**2. FAQ Rich Results OFFICIALLY RETIRED — May 7, 2026 (Breaking Change)**
Google confirmed FAQ rich results are gone as of May 7. The FAQPage schema markup is still valid (no errors at validator.schema.org) but produces zero SERP lift. Search Console API support ends August 2026. Any PSEO templates or client sites still relying on FAQ schema for SERP visibility need immediate audit.

**3. llms.txt Clarified: Not a GEO Ranking Signal — But a Real Agentic Tool (May 2026)**
Google's Search team explicitly stated llms.txt is not needed for AI search visibility. Meanwhile, the groundingpage.com canonical (updated May 28, 2026) documents the key semantic split: llms.txt as a GEO ranking lever has zero documented cross-platform visibility evidence. llms.txt as an agentic discovery/navigation file (its original intent) remains a legitimate, practical use case. Sites should not deploy llms.txt as a citation-boosting hack — deploy it only if your content needs to serve inference-time AI agents.

## Action Items for PSEO Network Template

- **Audit robots.txt** on all template-generated sites for AI crawler blocks. Cloudflare changed defaults to block AI bots — check if affected. Key bots: GPTBot, ClaudeBot, Claude-SearchBot, Google-Extended, PerplexityBot, OAI-SearchBot.
- **Remove or deprecate FAQPage schema** from the template's structured data library. Replace with FAQComponent if FAQ content is still needed for user experience — just don't expect SERP lift from it.
- **Add llms.txt generation** to the agentic-optimization checklist with the correct framing: agent navigation aid, not a GEO ranking signal.
- **Validate against schema.org v30.0** (released March 19, 2026). New types: `Credential`, `Error` (with `errorCode`), `floorLevel` (for LocalBusiness/Residence), `jobDuration` (for JobPosting).
- **JavaScript rendering check**: AI parsing fails 77% of the time for JS-rendered content. Audit whether dynamic content loads (pricing, features, comparisons) are server-rendered or client-side hydrates. This is a silent AI visibility killer.
