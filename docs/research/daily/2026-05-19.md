## 🔴 Breaking Changes

### 1. Google March 2026 Core Update Reshaped Structured Data Value
- **What happened:** FAQ rich results dropped ~47% post-update. How-To rich results vanished for supplementary content. Review schema on editorial comparison posts was demoted at scale. (Source: DigitalApplied.com, Mar 20, 2026)
- **Why it matters:** The update shifted schema's role from SERP display trigger → AI trust/entity verification signal. 3.2x citation lift in AI Mode for sites with clean entity schema.
- **Template impact:** PSEO templates must emphasize entity-based schema (Organization, WebSite, Article) over promotional types (FAQ, How-To) unless they describe primary page content.

### 2. Google AI Mode Links Updated (May 6, 2026)
- **What happened:** Google updated how links within AI Overviews & AI Mode are displayed and cited. (Source: YouTube, May 7, 2026)
- **Why it matters:** Zero-click search now at 93% of AI queries. Every citation in AI Mode is now more valuable—and more fragile.
- **Template impact:** Ensure citation-optimized content structure (clear attribution hooks, named entities, quoted facts) is in base template.

## 🟡 Notable Developments

### 3. Anthropic Claude Crawlers Now Require Granular robots.txt Handling
- **What happened:** Three distinct Claude crawlers: `ClaudeBot` (training), `Claude-User` (user requests), `Claude-SearchBot` (search indexing) — each needs separate robots.txt rules. (Source: Search Engine Journal, Feb 25, 2026)
- **Why it matters:** One-size-fits-all AI crawler blocking/allowing no longer works. PSEO sites need crawler-aware configuration.
- **Template impact:** Add robots.txt guidance section covering AI-specific crawler rules.

### 4. llms.txt Adoption Growing But Still Informal
- **What happened:** llms.txt files adopted for eCommerce and content sites to guide ChatGPT/Gemini. Google added llms.txt to its own site (Dec 2025). No formal standard yet. (Sources: Wix AI Search Lab, higoodie.com, YouTube Dec 2025)
- **Template impact:** Document llms.txt as a recommendation but flag it as non-standard.
