## Top 3 Findings

### 1. FAQ Rich Result DEPRECATED (May 7, 2026)
Google has officially deprecated the FAQ rich result feature as of May 7, 2026. Sites using `FAQPage` structured data will no longer receive rich results in Google Search. This is a breaking change for any PSEO template that auto-generates FAQ schema markup — that markup is now dead weight and may trigger spam signals if overused.

**Source:** [Google Search Documentation Updates](https://developers.google.com/search/updates) — May 8, 2026 entry

### 2. WebMCP Shipped in Chrome Early Preview (Feb 2026, gaining traction)
Google shipped WebMCP in Chrome early preview, allowing websites to expose structured, callable tools to AI agents via `navigator.modelContext`. This replaces fragile DOM scraping with a formal browser protocol. Sites that expose WebMCP endpoints will have privileged relationships with AI agents crawling and acting on their behalf. This is a fundamental shift toward the "agentic web."

**Sources:** [VentureBeat](https://venturebeat.com/infrastructure/google-chrome-ships-webmcp-in-early-preview-turning-every-website-into-a), [p0stman guide](https://p0stman.com/guides/agentic-web-webmcp-guide-2026), [BuildMVPFast](https://www.buildmvpfast.com/blog/webmcp-browser-standard-ai-agents-2026)

### 3. Google Preferred Sources Now Global (April 30, 2026)
Google expanded Preferred Sources to all languages and regions globally on April 30, 2026. Previously English-only, this feature lets users customize their "top stories" sources. For publishers, this means structured data and E-E-A-T signals are now directly tied to user preference mechanics across every market.

**Sources:** [Search Engine Land](https://searchengineland.com/google-preferred-sources-now-works-for-all-languages-476051), [9to5Google](https://9to5google.com/2026/04/30/google-preferred-sources-global-language-expansion/), [The Verge](https://www.theverge.com/tech/920985/googles-preferred-sources-feature-is-now-available-in-more-languages)

## Action Items for PSEO Network Template

1. **Remove or deprecate FAQPage JSON-LD** from auto-generated pages — FAQ schema is no longer supported as a rich result. Replace informational depth with Article + HowTo structured data where appropriate.
2. **Add WebMCP endpoint support** — Create a `/mcp.json` or similar endpoint exposing site tools in WebMCP format. This positions the template ahead of the agentic web curve.
3. **Audit robots.txt** for AI crawler access — with crawl-to-referral ratios collapsing, AI bots may be more aggressively blocked. Ensure `ChatGPT-User`, `Claude-Web`, `Bytespider`, `Google-Extended`, `PerplexityBot` have appropriate access if visibility matters.
4. **Reinforce E-E-A-T structured data** — with Preferred Sources now global, Organization, Author, and SpeakableSpecification markup are higher leverage than FAQ schema.
5. **Update "read more" deep link patterns** (April 20 addition to docs) — ensure content fragments are structured for snippet selection.
